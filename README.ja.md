# 自学サポートアプリ JIGAKU

JIGAKU は、小学生向けの無料の学習アプリシリーズです。ひらがな・カタカナ、時計、分数計算などの基本的な学習内容を、シンプルで広告のないフルスクリーンアプリで学習できます。

## 機能

- ひらがなとカタカナの練習
- カタカナとひらがなの練習 
- 時計の読み取り練習
- 足し算の練習
- 分数表現の練習
- 単位換算

## 必要環境

特に必要なものはありません。ブラウザさえあれば利用できます。

## 使い方

ブラウザで[JIGAKUのデモページ](https://code4fukui.github.io/jigaku/)にアクセスし、お好みのアプリを選んで使用してください。

## データ・API

アプリで使用しているデータは、CSV、JSON、HTMLで提供されています。

## ライセンス

このプロジェクトはクリエイティブ・コモンズ 表示 4.0 国際ライセンスの下に提供されています。
